## 檔案整理筆記 (File Organization)
為了保持結構清晰並方便偵錯與查閱，本儲存庫遵循以下整理原則：
- **一題一檔**：每一個題目（Problem/Exercise）都擁有自己獨立的 `.cpp` 原始碼檔案。
- **目錄分類**：檔案會依據週次以及題號分類存放於對應的資料夾中（如 `/Week01/UVa10035`, `/Week02/UVa272`）。
- **檔案命名**：檔案名稱將反映題號及功能（例如：`UVa10035_new.cpp`, `UVa10035_original.cpp`）。
- **檔案內容**：每一個`.cpp` 檔中都會有相關的思路註解。
